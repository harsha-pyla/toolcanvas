# ToolCanvas

ToolCanvas is a comprehensive, client-side suite of free, secure, and private online utility tools designed to simplify your daily digital tasks with zero signups or registration required.

## Key Features

All tools process data entirely within your web browser, ensuring complete privacy with zero server uploads:

*   **Social & Media Previews**: Custom builders for Instagram and LinkedIn profile drafts.
*   **Digital Visiting Card Creator**: Elegant single-page business card templates with real-time editing, custom styling, local PNG downloads, and dynamically generated QR codes.
*   **Infinite Drawboards**: Zoomable, pannable Infinite Whiteboard and Blackboard (Chalkboard Mode) supporting local coordinate links compression for instant sharing.
*   **Rich Text Notepad**: Notepad interface supporting ruled, legal, and grid paper textures, custom typography, autosaving, search & replace, and custom spellcheck systems.
*   **Client-Side Image Converters**: Batch processors for image resizing, format conversions (JPG, PNG, PDF, WebP), and Tesseract.js optical character recognition (OCR) extracting text locally.
*   **Random Generators**: Password manager, list shuffler, random place descriptor, and business name generator.
*   **Video Tools**: Client-side video trimming, metadata viewing, screenshot capturing, audio extraction (WAV), and thumbnail generator overlay designers.
*   **Resume Showcases**: ATS 90+ score optimized layouts for experienced executives and entry-level graduates.

## Privacy & Security

ToolCanvas operates on a serverless, client-side execution model. All calculations, image rendering, PDF compiling, and text recognition take place directly on your local device's CPU/GPU, ensuring absolute data confidentiality.
